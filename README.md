# 🏦 Bank System (Java Console Project)

A simple banking system built using Core Java and OOP concepts.

## 🚀 Features
- Create account
- Deposit money
- Withdraw money
- Check balance
- Menu-driven system

## 🧠 Concepts Used
- Classes & Objects
- ArrayList
- Constructors
- Scanner
- Loops & Switch case

## ▶️ Run
```bash
javac bankAprill/BankSystem.java
java bankAprill.BankSystem
